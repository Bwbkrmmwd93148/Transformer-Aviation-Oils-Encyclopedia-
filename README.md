## 🚀 البدء السريع

### 1. التثبيت
```bash
# استنساخ المشروع
git clone <repository-url>
cd abu_bakr_dynamic_site

# تثبيت المكتبات
npm install
```

### 2. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة البيانات
mysql -u root < config/database.sql

# أو عبر MySQL Workbench
```

### 3. إعدادات البيئة
```bash
# تحديث ملف .env
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=abu_bakr_oil_site
```

### 4. تشغيل الخادم
```bash
# في بيئة التطوير
npm run dev

# في بيئة الإنتاج
npm start
```

### 5. الوصول للموقع
```
http://localhost:3000
```
